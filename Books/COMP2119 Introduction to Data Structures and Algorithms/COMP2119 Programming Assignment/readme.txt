**COMP2119 Programming Assignment.cpp & COMP2119 Programming Assignment.exe
**This program reads an input file, input.txt
**Each line in the file refers to one operation
**0->Print the nodes of the AVL Tree in the inorder traversal
**1->Insert a node with key and string to the AVL Tree
**2->Delete a node with a given key
**3->Search a node with a given key and print both the key and the string
**
**
**The program was compiled in the below environment
**OS: Windows 10 Professonal 64-bit with Anniversary Update
**IDE: Dev C++ 5.11 64-bit
**Compiler: TDM-GCC MinGW 4.9.2 64-bit
**
**
**Sample input:
**0
**1 10 Apple
**1 50 Orange
**0
**2 10
**1 30 Banana
**3 50
**1 40 Mango
**0
**
**
**Sample ouput:
**-1
**10,Apple
**50,Orange
**50,Orange
**30,Banana
**40,Mango
